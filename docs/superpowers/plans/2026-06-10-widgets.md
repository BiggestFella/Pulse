# Widgets (WidgetKit) — implementation plan (BAK-19)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A single "Today's Workout" WidgetKit widget (five families) that renders from a shared App Group snapshot the app writes from the Today repository, deep-linking back into the app. Live Activity untouched.

**Architecture:** App maps `TodaySnapshot` → `WidgetSnapshot` (shared `Codable`), writes it to App Group `group.au.com.codeheroes.pulse` via `WidgetSnapshotStore`, and calls `WidgetCenter.shared.reloadAllTimelines()`. The extension's `PulseProvider: TimelineProvider` reads the snapshot (mock fallback), and `TodayWorkoutWidget` renders five families using the `Theme`/`Lockup`/week-cell pattern already compiled into the extension. Deep links (`pulse://start-today`, `pulse://today`) resolve in `AppShell.onOpenURL`.

**Tech Stack:** SwiftUI, WidgetKit (`TimelineProvider`, `Gauge`, `widgetURL`), App Groups (`UserDefaults(suiteName:)`), XcodeGen.

**Spec:** `docs/superpowers/specs/2026-06-10-widgets-design.md`.

> **CI reality:** the authoring environment has no Xcode toolchain — CI (macOS, simulator) is the build/test gate. WidgetKit *view* rendering isn't unit-testable here; cover the logic (mapping, store, provider entries, URL parsing) with `PulseWidgetsTests`/`PulseTests` and verify rendering manually. Gate after each code task: `xcodebuild -project Pulse.xcodeproj -scheme Pulse -destination 'platform=iOS Simulator,name=iPhone 17' -only-testing:PulseTests -only-testing:PulseWidgetsTests test`.

```swift
struct WidgetSnapshotStore {
    static let appGroup = "group.au.com.codeheroes.pulse"
    private static let key = "widget-snapshot"
    private let defaults: UserDefaults

    init(defaults: UserDefaults? = nil) {
        self.defaults = defaults ?? UserDefaults(suiteName: Self.appGroup) ?? .standard
    }
    func write(_ s: WidgetSnapshot) {
        if let data = try? JSONEncoder().encode(s) { defaults.set(data, forKey: Self.key) }
    }
    func writeRaw(_ data: Data) { defaults.set(data, forKey: Self.key) }   // test seam
    func read() -> WidgetSnapshot {
        guard let data = defaults.data(forKey: Self.key),
              let s = try? JSONDecoder().decode(WidgetSnapshot.self, from: data),
              s.hasValidWeek else { return .sample }
        return s
    }
}
```

func testMapsWorkoutDaySnapshot() {
    var reloaded = 0
    let store = WidgetSnapshotStore(defaults: UserDefaults(suiteName: "t.\(UUID())")!)
    let writer = WidgetSnapshotWriter(store: store, reload: { reloaded += 1 })
    writer.update(from: .sample, palette: .coastal)         // TodaySnapshot.sample
    let s = store.read()
    XCTAssertEqual(s.todayWorkoutName, "Chest & Tris")
    XCTAssertEqual(s.exerciseCount, 7)
    XCTAssertEqual(s.streak, 27)
    XCTAssertEqual(s.week.count, 7)
    XCTAssertEqual(s.startRoute, WidgetDeepLink.startToday.url.absoluteString)
    XCTAssertEqual(s.palette, "coastal")
    XCTAssertEqual(reloaded, 1)
}
